# PC Header 参考 Bitget 优化

## 背景

用户希望 PC 端 Header 可以参考 Bitget 中文首页的顶部导航。当前 PC Header 已经具备品牌 Logo、主导航、交易下拉、语言切换、登录/注册和用户入口，但视觉更偏霓虹风格，和参考站点的深色紧凑交易所导航存在差异。

## 范围

- 优化 `pc/src/components/layout/Header.vue` 的桌面 Header 结构与样式。
- 保留现有路由、登录态、语言切换、交易对行情下拉和交易对跳转逻辑。
- 现货入口继续跳转到现有 `Trade` 路由名，对应实际路径 `/spot/:symbol?`。
- 不新增后端接口，不修改业务数据结构。

## 设计依据

- Bitget 顶部导航为深色、低高度、左侧 Logo + 横向产品导航、右侧紧凑动作区。
- Header 应更像交易工具入口：密度更高、按钮更克制、下拉菜单更清晰。
- 当前项目已有 PairLogo、行情 ticker、i18n 和 Tailwind token，应优先复用。

## 验收标准

- Header 左侧仍只展示品牌 Logo，不显示平台名称文本。
- 主导航拥有 Bitget 风格的紧凑横向入口，交易下拉为清晰的产品分组 + 现货行情列表。
- 右侧语言、登录、注册和用户入口视觉统一，未登录与已登录状态均保留。
- PC 类型检查通过，新增/既有 Header 静态测试通过。
