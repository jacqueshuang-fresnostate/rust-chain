# 优化充值地址导入规则选择

## 背景

充值地址创建页已支持文件导入，但导入按钮位于地址明细区域，网络和支持币种位于上方规则区域，用户容易误以为导入文件本身不需要选择对应规则。

## 目标

- 将“导入文件”入口与网络、支持币种、初始状态放在同一个规则区域。
- 将“限定资产”在创建页文案调整为“支持币种”。
- 导入文件后仍沿用当前选择的网络和支持币种提交。
- 地址明细区域只保留地址行管理动作。

## 非目标

- 不改变后端 batch 接口。
- 不支持同一个导入文件内混合多个网络或多组币种规则。
- 不调整编辑充值地址弹窗。

## 验收标准

- “添加充值地址”SideSheet 中网络、支持币种、初始状态、导入文件按钮处于同一规则区域。
- 文件导入后提交 body 使用界面选择的 network 与 asset_symbols。
- 手动新增地址仍可正常提交。
- 相关前端测试、类型检查和 diff 检查通过。
