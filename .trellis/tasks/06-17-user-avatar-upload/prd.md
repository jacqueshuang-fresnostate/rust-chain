# 用户头像上传

## 背景

PC 端用户中心目前只显示默认账号图标，用户资料接口也没有头像字段。需要允许已登录用户上传头像，并在个人中心与头部用户入口展示。

## 范围

- 后端新增用户头像 URL 存储字段。
- 后端提供用户认证的头像上传接口，复用现有上传配置和图片校验。
- 上传成功后更新当前用户的头像地址，并在用户资料接口返回。
- PC 端用户中心提供头像触发上传入口，上传后刷新本地用户资料。
- Header 用户入口优先显示用户头像。

## 非目标

- 不重做后台上传配置页面。
- 不引入裁剪、压缩、头像审核等高级流程。
- 不影响管理员已有图片上传接口。

## 验收

- 已登录用户可以选择图片文件上传头像。
- 上传成功后用户资料里的头像 URL 持久化。
- 个人中心和 Header 能显示最新头像。
- 图片上传继续遵守后台配置的供应商、大小和 MIME 限制。
