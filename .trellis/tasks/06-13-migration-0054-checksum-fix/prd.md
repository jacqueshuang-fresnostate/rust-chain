# 修复已执行迁移 0054 checksum 冲突

## 背景

执行 `sqlx migrate run` 时报错：

```text
error: migration 54 was previously applied but has been modified
```

`0054_seed_country_codes.sql` 是已经执行过的国家代码种子迁移。后续国家配置本地名称与中文备注需求把 0054 的国家名称改成了本地语言名称，并新增了指向 0055 的说明行，导致 sqlx 校验已执行迁移的 checksum 时失败。

## 范围

- 恢复 `0054` 为最早的英文国家名称种子，不依赖 `remark` 字段。
- 保留 `0055` 负责把英文种子名称回填成本地语言名称，并新增中文备注。
- 更新迁移测试，明确 `0054` 是已执行的英文种子迁移，`0055` 是后续本地化回填迁移。

## 验收

- 已执行过 0054 的数据库不再因为本地 0054 内容改动而被 sqlx 拦截。
- 从空库顺序执行 0054 → 0055 时，最终国家配置仍能得到本地语言名称和中文备注。
