## 4. contract-swap-api 模块

### 4.1 ContractCoinController
#### base_url="https://www.hippoweb3.net/swap/"
**基础路径**: `/coin`

#### GET /coin/base-symbol
**功能**: 获取基币

**响应**:
```json
{
  "code": 0,
  "data": ["USDT"]
}
```

---

#### GET /coin/coin-info
**功能**: 获取合约信息

**请求参数**:
| 参数名 | 类型 | 传递方式 | 必填 | 说明 |
|--------|------|----------|------|------|

```json
{
  "code": 0,
  "data": {
    "id": "合约ID",
    "symbol": "交易对",
    "baseSymbol": "基币",
    "coinSymbol": "交易币",

    "sort": "排序",
    "minTurnover": "最小成交额",
    "maxLeverage": "最大杠杆",

  }
}
```

### 4.2 OrderController

**基础路径**: `/order`

#### POST /order/add
**功能**: 添加合约委托订单

**请求参数**:
| 参数名 | 类型 | 传递方式 | 必填 | 说明 |
|--------|------|----------|------|------|
| symbol | String | @RequestParam | 是 | 交易对符号 |
| price | BigDecimal | @RequestParam | 条件必填 | 委托价格(限价必填) |
| amount | BigDecimal | @RequestParam | 是 | 委托数量(张) |
| direction | Integer | @RequestParam | 是 | 方向(0:开多,1:开空,2:平多,3:平空) |
| type | Integer | @RequestParam | 是 | 类型(0:限价,1:市价) |
| leverage | BigDecimal | @RequestParam | 否 | 杠杆倍数 |

**响应**:
```json
{
  "code": 0,
  "message": "成功",
  "data": "订单ID"
}
```

---

#### POST /order/history
**功能**: 历史委托

**请求参数**:
| 参数名 | 类型 | 传递方式 | 必填 | 说明 |
|--------|------|----------|------|------|
| symbol | String | @RequestParam | 否 | 交易对符号 |
| status | Integer | @RequestParam | 否 | 状态 |
| pageNo | int | @RequestParam | 是 | 页码 |
| pageSize | int | @RequestParam | 是 | 每页条数 |

**响应**: Page<ContractOrderEntrust>

---

#### POST /order/current
**功能**: 当前委托

**请求参数**: 同history

**响应**: Page<ContractOrderEntrust>

---

#### POST /order/cancel
